How to Create a Selenium Automation Maven Project from Scratch

Learn how to set up a new Maven project for Selenium automation, step by step. This guide will walk you through every detail—from project creation to adding dependencies—so you can get started with confidence!

Step 1: Create a New Maven Project

  1. Open Your IDE (Eclipse or IntelliJ):
    • Eclipse: File > New > Project...
    • IntelliJ: File > New > Project...
  2. Select Maven Project Type:
    • In the project dialog, select Maven Project and click Next.
  3. Choose Archetype:
    • In the Archetype selection, search for quickstart.
      Select maven-archetype-quickstart (template for a simple Java project).
  4. Enter Maven Coordinates:
    • Group Id: e.g., com.example or rahulshettyacademy
    • Artifact Id: e.g., SeleniumFrameworkDesign
    • Version, Package: Leave default or customize as needed.
  5. Finish:
    • Click Finish. The IDE will generate a new Maven project with standard folder structure and pom.xml.
Tip: The Group Id is usually your company or domain (e.g., rahulshettyacademy), and the Artifact Id is the name of your specific project.

Step 2: Understand the Project Structure

SeleniumFrameworkDesign/
  ├── pom.xml
  └── src/
      ├── main/
      │   └── java/
      └── test/
          └── java/
  • pom.xml – Maven Project Object Model
  • src/main/java – Source code
  • src/test/java – Automation test scripts

Step 3: Add Selenium & TestNG Dependencies

  1. Open pom.xml
  2. Add Selenium Java Dependency:
    3. <dependency>
  <groupId>org.seleniumhq.selenium</groupId>
  <artifactId>selenium-java</artifactId>
  <version>4.35.0</version>
</dependency>
  3. Add TestNG Dependency:
    4. <dependency>
  <groupId>org.testng</groupId>
  <artifactId>testng</artifactId>
  <version>7.8.0</version>
  <scope>test</scope>
</dependency>
  4. Remove JUnit (Optional)
Note: Choose TestNG version per Java version: Java 8 → 6.14.3, Java 11+ → 7.x+

Step 4: Your Project is Ready!

  • You now have a ready-to-use Maven project configured for Selenium automation with TestNG.
  • Add tests under src/test/java, package them, and start building automation.
The beauty of Maven: Just specify GroupId and ArtifactId, and Maven resolves all transitive dependencies!

Summary

  1. Create Maven Project using maven-archetype-quickstart
  2. Fill GroupId and ArtifactId
  3. Add Selenium + TestNG
  4. Remove JUnit
  5. Sync Maven
  6. Write your scripts!
