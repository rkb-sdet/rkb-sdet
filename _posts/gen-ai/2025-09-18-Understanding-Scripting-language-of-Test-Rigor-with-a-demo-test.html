Live Mode in testRigor: Automate End-to-End Flows in Plain English

Live Mode in testRigor empowers testers to build, validate, and debug automation flows step-by-step using plain English conventions, all directly from the browser UI—without writing code or specifying selectors.

How Does Live Mode Work?

  • From your TestSuite, launch Live Mode to open your application homepage.
  • Directly enter English instructions that mimic real user actions, such as:
    click "ADD TO CART" under Carrot
    click "cart"
    click "PROCEED TO CHECKOUT"
    enter "rahulshettyacademy" into field
    click "Apply"
    check that page contains "Code applied ..!"
  • The Live Mode UI shows real-time results, screenshots, and feedback after each executed step, making it easy to adjust or debug as you go.
  • You can reference elements contextually (using under, next to, below), making interactions robust even when UI has duplicate buttons or controls.

Hands-On Example: Coupon Flow Automation

  1. Add a Broccoli and a Carrot to the cart by referencing under item context in your click commands.
  2. Open the shopping cart and proceed to checkout with unique commands referencing visible labels.
  3. Enter coupons or test values into the promo code field, leveraging context such as to the left of Apply for ambiguous entry boxes.
  4. Apply the coupon and assert successful or failed application using check that page contains "Code applied ..!" or "Invalid code ..!".
  5. Clear fields (using keyboard inputs like ctrl+A and delete) and test negative scenarios seamlessly.

Key Features for Scalable Automation

  • Reusable rules: Write common steps once as Rule (e.g., login), then invoke by name in all tests.
  • Data-driven support: Test Data management lets you inject dynamic, external data into flows the same way as traditional frameworks.
  • Modern reporting: Failures are grouped by root cause, impacted tests are summarized, and screenshot comparisons highlight before/after UI changes for precise debugging.
  • Jira integration and fix propagation: One-click bug ticket creation and global fix/replace for repetitive failures.

Summary

AI-powered Live Mode transforms plain English into resilient, end-to-end automation in minutes—enabling testers of all skill levels to accelerate quality and feedback with minimal code and maximum productivity.
